# DK Bot 접기/펼치기 (헤더 바 최소화) — 설계

날짜: 2026-07-12
상태: 승인됨 (사용자 확인)

## 문제

DK Bot 패널(420×720 고정 오버레이)이 화면을 가려서, 뒤 내용을 보려면 매번 완전히 닫고(FAB로 복귀) 다시 열어야 한다. 대화는 보존되지만 왕복 클릭이 불편하다.

## 결정

패널 헤더에 접기 버튼을 추가해, 하단 우측의 **한 줄 알약형 바**로 최소화한다. 바를 클릭하면 즉시 다시 펼쳐진다. (대안이었던 "높이 축소", "반투명 토글"은 화면 가림이 남거나 가독성이 떨어져 기각.)

## 상태 모델

`DkBot`에 `collapsed: boolean` 상태 하나 추가. 시각 상태 3가지:

| 상태 | 렌더 |
|------|------|
| 닫힘 (`!open`) | 로봇 FAB — 기존 그대로 |
| 열림+펼침 (`open && !collapsed`) | 기존 패널 + 헤더에 접기 버튼(`ChevronDown`) — 버튼 순서 ⌄ ↺ ✕ |
| 열림+접힘 (`open && collapsed`) | 알약형 바: 로봇 마스코트 + "DK Bot" + `ChevronUp`. 헤더와 같은 `gradient-dark` 토큰 재사용 |

## 동작 규칙

- 접힌 바 전체가 하나의 버튼 — 클릭 시 펼침. 펼칠 때 입력창 포커스 + 대화 맨 아래로 즉시 스크롤(패널 재마운트 대응).
- 대화·입력·스트리밍은 접어도 전부 유지 (컴포넌트는 항상 마운트 — 기존 close/reopen 보존과 동일 메커니즘).
- 스트리밍(`loading`) 중 접혀 있으면 바에 pulse 점 표시로 진행 중임을 알림.
- Esc는 기존대로 완전 닫기(접힘 상태에서도 동일).
- X로 완전히 닫으면 `collapsed` 리셋 — 다음에 열 때는 펼친 상태로 시작.
- 접힘 상태는 세션 메모리에만 유지. 서버 동기화(user prefs) 안 함 — "잠깐 뒤 보기" 용도의 일시 상태.

## 변경 파일
